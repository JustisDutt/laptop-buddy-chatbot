# LaptopBuddy Chatbot

This project is a simple chatbot that helps users choose a laptop.

## Scenario
Assisting a customer with choosing the right product.

## How to run
1. Open the `web` folder
2. Open `index.html` in a browser

## Features
- Step-by-step conversation
- Error handling for invalid inputs
- Clear recommendation output

## Error handling examples
- Non-numeric budget input
- Invalid option selections

## Improvements
- Smarter intent detection
- Real product database
- Better UI
